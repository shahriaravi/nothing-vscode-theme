# Contributing to nothing-vscode-theme

Thank you for your interest in contributing! To help maintain quality, please follow these guidelines:

## How to Contribute

- **Fork** the repository
- **Clone** your fork and create a new branch:
  ```
  git checkout -b feature/your-feature-name
  ```
- **Commit** your changes with descriptive messages.
- **Push** to your fork and submit a Pull Request.

## Reporting Issues

- Use the Issues tab to report bugs or request features.
- Please include as much detail as possible (screenshots, steps to reproduce, etc.).

## Code Style

- Please follow [VSCode Theme Guidelines](https://code.visualstudio.com/api/extension-guides/color-theme).
- Make sure your code is readable and properly formatted.

## Additional Help

For questions, start a Discussion or use Issues.

Thank you for contributing!
